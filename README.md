## 概要

`Golang`と`React`を使った簡単な掲示板アプリ

## Contains

### API

- Golang
- Gin
- GORM
- goose
- Mysql
- Toml

### FRONT

- Typescript
- React.js
- Next.js
- styled-componets
- Semantic-UI

## usage

### build & start

```
make start
```

Per default the development server is listening on http://localhost:3000.

### run

```
make up
```

### stop

```
make down
```
